## 2. ロードマップ V2(V1 08章 §3 の差し替え。起点=2026年7月)

### Phase 1: 0〜3ヶ月(2026年7〜9月)— 基盤+記事エンジン点火

- P2〜P5 を完了。**記事30本**(C4悩み10・C2資格10・C5施設10)公開。
- 監修パートナー2名の契約(**ドリル着手の前提条件**。予算は H-32: 月額2〜5万円で見積り直し)。
- T-0c/T-0d 実施(クエリリストは V2 構成: ドリル系・給料系・比較系を追加)。
- マイルストーン: インデックス正常/GSC表示回数の右肩上がり/初の被リンク。

### Phase 2: 3〜6ヶ月(2026年10〜12月)— 給料DB第1弾+試験期を「事実データ」で迎える

- P6(給料エンジン第1弾+計算機)、P7(診断+301)。記事累計70〜80本。
- `/shiken/` ハブ+合格点推移+第38回試験の直前・直後コンテンツ(日程・持ち物・ボーダー考察 — **転載不要の事実データのみ**)で初回試験期(2027年1月)の需要を小さく受ける。
- ASP提携(転職3+講座2)。
- マイルストーン: 初CV/計算機への被リンク1件以上/試験期の実測クエリデータ取得。

### Phase 3: 6〜12ヶ月(2027年1〜6月)— ドリル本格投入+pSEO第2弾

- P8 ドリルエンジン公開: 介護福祉士3科目(論点60〜80)から開始し、**2027年内に13科目・論点300を完成**(H-33 のペースで分割投入)。
- P9: 47都道府県給料ページ(3〜4月の統計新年度公表に合わせて投入)+資格×都道府県。
- 記事累計150本。リライト運用開始。診断集計の独自調査公開(被リンク施策 — V1 R-09 対応を継続)。
- マイルストーン: 月10〜15万PV/月間CV10件/ドリル完走ユーザーのLINE登録開始。

### Phase 4: 12〜36ヶ月 — 全科目完成と100万PVへの積み上げ

- ケアマネドリル(2027年10月試験照準)・模試・施設比較マトリクス・記事500本へ増強。
- 2028年1月試験期: ドリル+速報系で試験期ピーク(H-27 の上限検証)。
- **目標: 24〜36ヶ月(2028年7月〜2029年7月)で月間100万PV**【H-31】。
