## §4. シンコンテンツゲート(全 pSEO 共通の公開条件 — R-12 対応)

1. **データ充足の足切り**: 対象軸の主要数値の null 率が30%を超える組み合わせはページを生成しない(V1 06章の「捏造禁止・null許容」を継承)。
2. **独自ブロック必須**: 統計値の表示だけのページは公開しない。全国比・前年差・関連ガイド3本への文脈リンク・グラフ(SVG)・編集部の解説段落(最低200字)を必ず含める。
3. **段階公開+インデックス率ゲート**: 1つの pSEO 群は50ページ単位で公開し、直前公開分のGSCインデックス登録率90%未満なら次を止める(V1 04章の公開ゲートを pSEO に拡張)。
4. **需要ゼロ軸の禁止**: 優先S/Aの群も、T-0c 実測で主要クエリの検索需要が確認できない軸は保留に落とす。
5. **カニバリ防止**: pSEO ページと記事の対象クエリ重複はキーワード管理表で突合(V1 06章 §4 を継承)。
