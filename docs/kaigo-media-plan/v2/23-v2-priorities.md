## 2. 月次時間予算(1人+AI・週15〜20h = 月65〜87h)【新設 — パネル指摘 M-01】

Phase 3(最繁忙期)の想定。**この表を超える計画は立てない**(超過見込み時はドリル→pSEO→記事の順に減速する)。

| 区分 | 内訳 | 月間時間 |
| --- | --- | --- |
| 記事制作 | 月12本 × 60〜90分(ai-company 検品込み・V1 H-08) | 12〜18h |
| ドリル制作 | 月15論点 × 60〜90分(H-33: AI生成+検証+類似チェック+監修往復) | 15〜23h |
| 監修往復・検品 | 監修コメント反映・差し戻し対応 | 3〜5h |
| pSEO 独自ブロック | 月50ページ × 10〜15分(生成+数値整合検品) | 8〜13h |
| 開発 | P8/P9 の残作業・ツール改修 | 5〜8h |
| リライト・更新 | 月4〜8本+更新カレンダー対応 | 4〜8h |
| 運用 | LINE配信確認・ASP管理・計測レビュー・独自調査 | 3〜5h |
| **合計** | | **50〜80h**(上限87hに対し残余7h〜) |

- 上限側は容量にほぼ張り付く。**H-33 が上振れした場合(1論点90分超)はドリルペースを月10論点に落とす**(完成時期が後ろ倒しになることを許容する — ドリルは「効かなければ縮小できる」構造)。
- 20章 §2-3 の「開発工数 約100〜110h」とは別に、**ドリルの人的工数は300論点 × 60〜90分 = 累計300〜450h**が存在する(この表で16〜20ヶ月に分割)。

## 3. ロードマップ V2(V1 08章 §3 の差し替え。起点=2026年7月)

### Phase 1: 0〜3ヶ月(2026年7〜9月)— 基盤+記事エンジン点火

- P2〜P5 を完了。**記事30本**(C4悩み10・C2資格10・C5施設10)公開。
- 監修パートナーの契約(**ドリル着手の前提条件**。予算は H-32: 1問単価300〜800円・月3〜10万円で見積り)。科目別体制(医療系スポット監修含む)は26章 §5 工程5 に従う。
- T-0c/T-0d 実施(クエリリストは V2 構成: ドリル系・給料系・比較系を追加。**ドリル系はSERP意図分析まで行い P8 のゲート判定に使う**)。
- マイルストーン: インデックス正常/GSC表示回数の右肩上がり/初の被リンク。

### Phase 2: 3〜6ヶ月(2026年10〜12月)— 給料DB第1弾+試験期を「事実データ」で迎える

- P6(給料エンジン第1弾+計算機)、P7(診断+301)。記事累計70〜80本。
- `/shiken/` ハブ+合格点推移+**第39回試験(2027年1月)**の直前・直後コンテンツ(日程・持ち物・ボーダー考察 — **転載不要の事実データのみ**。タイトルは「合格点・ボーダー予想」— 22章 R-11)+過去問情報ハブ(25章 #111〜113)で初回試験期の需要を小さく受ける。
- ASP提携(転職3+講座2)。**提携時にLINE配信内アフィリンクの可否を案件ごとに確認**(不可なら2段構成 — 26章 §7)。
- マイルストーン: 初CV/計算機への被リンク1件以上/試験期の実測クエリデータ取得。

### Phase 3: 6〜12ヶ月(2027年1〜6月)— ドリル本格投入+pSEO第2弾

- P8 ドリルエンジン公開: 介護福祉士3科目(論点40〜60)から開始し、**月15論点前後のペースで16〜20ヶ月かけて13科目・論点300を完成(完成目標: 2028年前半)**(初稿の「2027年内完成」は監修工数の検算で不成立のため修正 — §2)。
- P9: 47都道府県給料ページ(3〜4月の統計新年度公表に合わせて投入)+資格×都道府県。
- 記事累計150本。リライト運用開始。診断集計の独自調査公開(被リンク施策 — V1 R-09 対応を継続)。
- マイルストーン: 月10〜15万PV/月間CV10件/ドリル完走ユーザーのLINE登録開始/**ドリル公開3ヶ月で検索流入 月3千セッション以上(中間閾値 — 未達なら §4 の18ヶ月判定を待たず投入ペースを見直す)**。

### Phase 4: 12〜36ヶ月 — 全科目完成と100万PVへの積み上げ

- ドリル300論点完成(2028年前半)・ケアマネドリル(2027年10月試験照準※複択形式派生)・模試・施設比較マトリクス・記事500本へ増強。
- 2028年1月試験期: ドリル+試験事実データで試験期ピーク(H-27 の上限検証)。
- **目標: 24〜42ヶ月で月間100万PV**【H-31】(現行仮説の上限和は約80万PV — 22章 §1。100万には記事800本への増強・pSEO第2弾・ケアマネ/模試の上積みが必要。12ヶ月目の再推計で経路を確定する)。
