## V2 の結論サマリー(3行)

1. PVの主砲は「過去問1,600ページ」から **「オリジナル学習ドリル × 統計DB(pSEO) × 記事500本」の分散型** へ移行する。試験期スパイク依存を下げ、Googleアップデント耐性と著作権安全性を上げる。
2. **収益への影響は軽微**: V1 でも過去問70万PVは収益式から除外していた(CV対象=非過去問30万PV)。V2 では CV対象PVがむしろ増える(60〜90万PV【仮説】)。失うのはPVの見栄えと LINE獲得の一部であり、売上の柱ではない。
3. 100万PV到達の見立ては 18〜30ヶ月 → **24〜36ヶ月に後ろ倒し**(試験期の瞬間風速に頼れないため)。代わりに通年PVの比率が上がり、恒常化はむしろ早まる可能性がある。
