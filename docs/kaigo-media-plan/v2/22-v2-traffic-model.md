### 合計(試験期ピーク月)

| シナリオ | 合計PV |
| --- | --- |
| 全仮説が下限 | 約48〜58万 |
| 中央値 | **約75〜90万** |
| 全仮説が中〜上位 | 約100〜125万 |

- **月間100万PVは「中〜上位シナリオ×コンテンツ完成(記事500本+ドリル全科目+pSEO第2弾)」で達成する目標**であり、V1 より難度は上がる。到達見立て: **24〜36ヶ月【H-31】**(V1 の18〜30ヶ月から後ろ倒し)。
- 一方で**試験期スパイクへの依存度が下がる**(V1: PVの70%が12〜1月集中 → V2: 試験期依存は20〜25%)。恒常100万PVの達成はむしろ V1 より近い。収益は通年で平準化する。

## 4. 収益モデル V2(V1 07章 §4 の基数を差し替え)

```
成熟時 月間85万PV(中央値シナリオ)のうち CV対象PV(記事+給料+資格+ツール)= 55〜70万
× アフィリンク到達ページ比率 60%【H-24 継続】
× クリック率 1.5〜3%【H-05 継続】
× CV率 3〜8%【H-06 継続】 × 承認率 30〜70%【H-23 継続】 × 単価5,000〜15,000円【H-07 継続】
= 中央値シナリオ: 月80〜200万円【H-36】(V1 の60〜150万から上方。CV対象PVの構成比が30%→70%に上がるため)
```

- **V2 の本質的な改善点**: V1 は「PVの70%(過去問)がほぼ収益化されない」構造だった。V2 は PVの大半がCV対象クラスタに乗るため、**同じPVなら収益性が高い**。100万PVの達成が遅れても収益マイルストーンは維持できる見込み(H-36 で管理)。
