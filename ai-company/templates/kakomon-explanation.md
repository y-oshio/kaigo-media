# 過去問解説テンプレート(F1規格 — 設計書04章 §2)

> 過去問パイプライン(記事パイプラインとは別系統)で使用。データは `KakomonQuestion` スキーマ(設計書06章)に格納され、このテンプレートは explanation フィールドの品質規格を定める。

## 構成(1問分)

```
問題文(原文どおり。改変禁止)
選択肢1〜5(タップで正誤表示。正解・解説は初期HTMLに含める — 設計書05章 §4)
正解: N

【解説】← このページの存在理由。定量下限は絶対に割らない
- why(300字以上): なぜこの選択肢が正解か。根拠となる制度・法令・知識体系名を最低1つ明記
- whyNotOthers(各選択肢1文以上): 選択肢1は〜のため誤り。/ 選択肢2は〜
- tips(任意): 覚え方・頻出パターン
- lawUpdateNote(該当時必須): 「出題当時は〜だったが、現行制度(20XX年改定)では〜」

出典表記: 第N回介護福祉士国家試験 問N(公益財団法人社会福祉振興・試験センター)
関連問題: 同一科目3問への内部リンク
```

## 品質基準(CI機械チェック対象)

- [ ] why が300字以上
- [ ] whyNotOthers が選択肢数と同数(正解を除く)
- [ ] 制度・法令名が最低1つ含まれる
- [ ] correctIndex が公式正答表と人手突合済み(status: fact-checked 以上)
- [ ] 制度改定の影響を fact-checker が確認済み(改定があれば lawUpdateNote 必須)
- [ ] 第38回以降はパート(A/B/C)情報が subject マスタに設定済み

## 禁止事項

- 問題文・選択肢の要約・言い換え(原文主義)
- 「解説: 正解は3です。」だけの空解説(下限割れは公開不可)
- 他サイト解説の転用・パラフレーズ
