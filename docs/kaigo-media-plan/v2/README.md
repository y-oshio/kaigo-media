## V2 の結論サマリー(3行)

1. PVの主砲は「過去問1,600ページ」から **「オリジナル学習ドリル × 統計DB(pSEO) × 記事500本」の分散型** へ移行する。試験期スパイク依存を下げ、Googleアップデート耐性と著作権安全性を上げる。
2. **収益への影響は軽微**: V1 でも過去問70万PVは収益式から除外していた(CV対象=非過去問30万PV)。V2 では CV対象PVが成熟時30〜50万【仮説】に増え、**PVに占めるCV対象の構成比が約30%→約73%に上がる**(22章 §4)。失うのはPVの見栄えと LINE獲得の一部であり、売上の柱ではない。
3. 100万PV到達の見立ては 18〜30ヶ月 → **24〜42ヶ月に後ろ倒し**(現行仮説の上限和は約80万PV。100万には Phase 4 拡張〔記事800本・ケアマネ/模試・pSEO第2弾〕が必要 — 22章 §1)。代わりに通年PVの比率が上がり、収益は平準化する。
