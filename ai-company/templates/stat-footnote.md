# 統計引用・脚注テンプレート(F2規格 — 設計書04章 §2)

給料・統計に触れる全ページで使用する定型文。**数値と定義を切り離して書かない**こと。

## 平均給与額(介護従事者処遇状況等調査)を引用する場合

```
※「平均給与額」は基本給(月額)+手当+一時金(4〜9月支給金額の1/6)の合計で、
介護職員等処遇改善加算を取得している事業所の常勤職員の値です。
出典: 厚生労働省「令和N年度介護従事者処遇状況等調査」(確認日: YYYY-MM-DD)
```

## 賃金構造基本統計調査を引用する場合

```
※「きまって支給する現金給与額」は残業代等を含む税引き前の月額で、
上記の「平均給与額」(処遇状況等調査)とは定義が異なるため直接比較できません。
出典: 厚生労働省「賃金構造基本統計調査」(調査年: N年、確認日: YYYY-MM-DD)
```

## 禁止パターン

- 異なる調査の数値を同一の表・同一の文で比較する(定義差の説明なしの並記は誤解を生む)
- 「厚労省調べ」だけの出典表記(調査名・年・確認日の3点セットが必須)
- 全国値を都道府県の値であるかのように書く(粒度の偽装)
- 出典に存在しない粒度の数値を推計で補う(データが無い場合は「公表データなし」と書く)

## 更新ルール

統計の新年度公表時(処遇状況等調査: 例年3〜4月)は、`data/` の SalaryStat を surveyYear 追加で更新し(上書き禁止 — 履歴保持)、影響ページの数値・年号・確認日を一括更新する(設計書R-08: 公表後30日以内SLA)。
